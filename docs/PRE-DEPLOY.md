# Before you deploy CSS (optional)

Only worth a glance if you changed **header** or **footer** layout.

Quick check: open `/` and one interior page (e.g. `/product-team-AI-vibe-coding`) at ~800px width — nav should match (centered links, Journal on the right, title not under the bar).

**CSS rule:** don’t scope global `header` to `body.page-home` only. There’s a short comment in `layout.css` at the header block.
