Mikes-Website
=============
A copy of my simple website

## Local dev

```bash
php -S 127.0.0.1:8000 router.php
```

Open http://127.0.0.1:8000 (or http://localhost:8000). Use `127.0.0.1` in the bind address so the server accepts IPv4; `localhost` alone can listen only on IPv6 and the browser may refuse the connection.
